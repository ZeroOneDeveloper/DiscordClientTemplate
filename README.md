# Discord Client Template (Cogs Structure)

## Test Environment
- macOS Sonoma, Version 15.1 Beta (24B5055e)
- Python 3.12.6 64-bit
- discord.py 2.3.2

## Installation
```shell
pip3 install -r requirements.txt
```
Create an `.env` file.
```dotenv
TOKEN=Your client token
```

## Usage
```shell
python3 .
```
